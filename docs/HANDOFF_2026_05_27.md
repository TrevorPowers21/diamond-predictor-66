## Core data model

| Concept | Where it lives |
|---|---|
| Players master roster | `players` table (FK target for everything else) |
| Hitter stats | `Hitter Master` (current season + history) |
| Pitcher stats | `Pitching Master` |
| Per-pitch shape data | `pitcher_stuff_plus_inputs` (per pitch type per hand) |
| Conference env+ | `Conference Stats` |
| Park factors | `"Park Factors"` (uppercase, the canonical one) |
| Team aggregate WAR | `team_war_snapshots` (now includes `national_seed_rank`) |
| Projections | `player_predictions` |
| Coach watchlist | `high_follow` |
| Coach roster build | `team_builds` + `team_build_players` |
| Per-team override | `target_board` |

**Division values:** `D1` or `NJCAA_D1`. JUCO is excluded from most user-facing displays + AI scouting reports.

**Prediction variants:**
- `model_type=returner` + `variant=regular`, `customer_team_id=NULL` → global cross-team projection
- `model_type=transfer` + `variant=precomputed`, `customer_team_id=<uuid>` → per-customer-team projection (the "team you're impersonating sees their own numbers" feature)

**Seasons:**
- `CURRENT_SEASON = 2026` (the season just played)
- `PROJECTION_SEASON = 2027` (what's projected — where predictions live)
