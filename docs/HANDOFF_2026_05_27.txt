================================================================
WHAT RSTR IQ IS
================================================================

AI-powered college baseball player development platform. Coaches use it
to scout, project, and value players for roster building + transfer
portal decisions. Per-team subscription. 8 customer teams currently:
Arizona State, Arkansas, Florida Atlantic, Georgia, Kansas, Penn State,
Stetson, TCU.

Stack: Vite + React + TypeScript + shadcn/ui + Tailwind on Vercel.
Backend Supabase (Postgres + Edge Functions + Auth + Storage). AI:
Anthropic Claude (Haiku 4.5 for bulk, Sonnet 4.6 planned for live
copilot).

Local repo: /Users/danielleogonowski/dev-main/diamond-predictor-66

User: Trevor (trevor.m.powers21@gmail.com). Baseball coach / program
leader. Sharp, direct, results-oriented. Don't over-explain or hedge.


================================================================
CORE DATA MODEL
================================================================

players                       — master roster, FK target
Hitter Master                 — hitter stats (current + history)
Pitching Master               — pitcher stats
pitcher_stuff_plus_inputs     — per-pitch shape data per hand
Conference Stats              — conference env+
"Park Factors"                — canonical park factors (uppercase!)
team_war_snapshots            — team aggregate WAR (now incl. national_seed_rank)
player_predictions            — all projections
high_follow                   — coach watchlist
team_builds + team_build_players  — coach roster builds
target_board                  — per-team player override

Division values: 'D1' or 'NJCAA_D1'. JUCO excluded from most user-facing
displays + AI scouting reports.

Prediction variants:
  - returner/regular + customer_team_id NULL → global cross-team
  - transfer/precomputed + customer_team_id set → per-customer-team
    ("team you're impersonating sees their own numbers")

Seasons:
  - CURRENT_SEASON = 2026 (the season just played)
  - PROJECTION_SEASON = 2027 (where predictions live)
